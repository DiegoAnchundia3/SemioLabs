# SindroGames Frontend

Frontend base en React + TypeScript para SindroGames.

## Scripts

- `npm run dev` para desarrollo
- `npm run build` para compilar
- `npm run preview` para previsualizar la build

## Estado actual

- Pantalla principal con acceso a estudiar y practicar
- Secciones visuales para usuario, puntuación, rankings y modo estudio
- Backend pendiente para autenticación, salas y persistencia
